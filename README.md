# Student-Record-Management-System-C
Console-based Student Record Management System in C using Linked Lists, Dynamic Memory Allocation, File Handling, CRUD operations, and persistent data storage.

## Features

- Add Student Records
- Display Student Records
- Search Students by University Roll Number
- Update Student Information
- Delete Student Records
- Sort Records by University Roll Number
- Unique ERP ID Validation
- Unique Roll Number Validation
- Automatic Data Persistence
- Automatic File Creation and Deletion
- Menu-Driven Course and Branch Selection

## Technologies Used

- C Programming
- Linked Lists
- Dynamic Memory Allocation
- File Handling
- Dev-C++

## Student Attributes

- Course
- Branch
- University Roll Number
- ERP ID
- Name
- Phone Number

## Data Storage

Records are stored locally in:

`students.txt`

The file is automatically:
- Created when the first record is added
- Updated when records are modified
- Deleted when all records are removed

## Concepts Demonstrated

- Data Structures and Algorithms
- Linked List Operations
- CRUD Operations
- File Handling
- Searching
- Sorting
- Memory Management
